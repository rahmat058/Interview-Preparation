## Recommended Study Path

### Week 1 — Foundations

1. **01** — What Docker is, containers vs VMs, image vs container
2. **05** — Run commands daily on a sample Node/MERN app

### Week 2 — Build & Run

3. **02** — Write a Dockerfile for Express API + multi-stage React build
4. **03** — Docker Compose for MERN (app + MongoDB + Redis)

### Week 3 — Production

5. **04** — Security, registries, CI/CD pipeline
6. Build and push one image to Docker Hub

### Interview Week

7. Skim **Interview Answer** boxes in all files
8. Practice: `docker build`, `docker compose up`, debug with `docker logs` and `exec`
