## Vue vs React Folder

| React | Vue equivalent |
|-------|----------------|
| useState | ref / reactive |
| useEffect | watch / watchEffect / onMounted |
| Custom hooks | Composables |
| Context | provide/inject, Pinia |
| Redux | Pinia |
| React Query | TanStack Vue Query |
| Next.js | Nuxt 3 |
| JSX | Templates + script setup |

---

## Stack Assumed

- **Vue 3** with Composition API + `<script setup>`
- **Pinia** (not Vuex for new projects)
- **Vue Router 4**
- **Vite** or **Nuxt 3**
- **TypeScript** where noted
- **Vitest** + Vue Test Utils for testing
